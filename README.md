# Guitar Songs Manager

Guitar Songs Manager is a FastAPI app with a single-page UI for storing guitar
songs, lyrics, and chords. It keeps data in SQLite, detects verse/chorus
structure, and helps propagate chords across repeated sections.

Features:
- Create, edit, and view songs with lyrics and chords.
- Auto-detect verses/choruses and optionally expand choruses in views.
- Chord editor that previews and propagates chord placements.

AI assistance: This project was created with assistance from AI agents.
